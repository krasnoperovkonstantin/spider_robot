Робат паук игрушка

- arduino pro mini
- аккумулятор 18650
- повышающий преобразователь на 5В - 2 шт.
- сервоприводы Micro Servo 9g SG90 - 8 шт.
- 3D печать
- любой ИК пульт (конфигурируется в прошивке)
- ИК приемник
